# dotfiles

My personal configuration files (dotfiles) for macOS, GNU/Linux, and Windows.


## Contents

The repository is split into two directories:

- `scripts/`: Shell scripts. Most of this stuff is outdated, but I keep it around for reference.
- `settings/`: Configuration files for various software tools (e.g., Zsh, Firefox).
