# dotfiles

dotfiles is a collection of configuration files for my personal setup on MacOS, GNU/Linux and Windows.

![Retro computer with an abstract image of a female robot on the screen](/assets/hero.jpeg)

Image generated using OpenAI's [DALL-E 3](https://openai.com/dall-e-3).


## Project Structure

The repository is organized as follows:

- `scripts`: Contains shell scripts. Most of this stuff is outdated, but I keep it around for reference.
- `settings`: Contains configuration files for various software tools (e.g., Zsh, Firefox).
